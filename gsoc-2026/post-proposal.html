Post-Proposal Development & Verification

  • BaseJump STL Verification (PR #743) ACTIVE REVIEW

    Developed a high-fidelity "UVM-lite" verification suite for bsg_fifo_1r1w_small. This work serves as the primary reference for modular verification in the bespoke-silicon-group ecosystem, utilizing SystemVerilog classes for Driver, Monitor, and Scoreboard logic without the overhead of the full UVM library, ensuring total compatibility with Verilator.

  • Ready-Valid-Yumi Protocol Mastery COMPLIANT

    Engineered the testbench to strictly adhere to the BSG Standard Timing specifications. This involved:

    • Implementing the v -> r (valid-yumi) consumer handshake and r -> v producer handshake.
    • Utilizing SystemVerilog Clocking Blocks to eliminate simulation race conditions during signal sampling.
    • Validating the 1-cycle "show-ahead" latency logic required for data-path performance.
  • Verification Automation & Diagnostics VERILATOR 5.030

    Developed an automated C++ test harness that executes high-volume randomized transaction sets. Integrated a queue-based model_q scoreboard to perform real-time data integrity checks against the RTL DUT.
